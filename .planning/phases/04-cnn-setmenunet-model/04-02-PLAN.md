**7 test functions:**

1. `test_shape_smoke()` -- CSMNET-06:
   - Create CNNSetMenuNet with default args.
   - `grid = torch.randn(4, 2, 11, 11)`, `capacity = torch.randn(4, 4)`, `options = torch.randn(4, 10, 6)`, `mask = torch.ones(4, 10, dtype=torch.bool)` with `mask[0, 7:] = False`.
   - `model.eval()`, forward with no_grad, assert output shape is (4, 10).

2. `test_encoder_output_dim()` -- CSMNET-02:
   - Create CNN_Encoder(dim=11, n_layers=2, n_filters=32, dropout=0.1, aux_dim=4).
   - `grid = torch.randn(4, 2, 11, 11)`, `capacity = torch.randn(4, 4)`.
   - Forward with no_grad, assert output shape is (4, 128).

3. `test_warm_start()` -- CSMNET-05:
   - Create CNN_2d(dim=11, n_layers=2, n_filters=32, dropout=0.1, aux_dim=4, output_dim=3).
   - Create CNNSetMenuNet(). Get CNN_2d state_dict via `cnn2d.state_dict()`.
   - Call `model.load_cnn_weights(cnn2d_state_dict)`.
   - Verify encoder conv1 weights match: `assert torch.equal(model.encoder.conv1.weight, cnn2d.conv1.weight)`.
   - Verify fc3 was NOT loaded (fc3 doesn't exist in CNN_Encoder): check that encoder state_dict has no 'fc3' key.
   - Print the warm-start log output and verify it mentions loaded/skipped counts.

4. `test_permutation_invariance()` -- CSMNET-04 (permutation invariant due to no positional encoding):
   - Create CNNSetMenuNet, model.eval().
   - `grid = torch.randn(2, 2, 11, 11)`, `capacity = torch.randn(2, 4)`, `options = torch.randn(2, 8, 6)`, `mask = torch.ones(2, 8, dtype=torch.bool)`.
   - Generate independent random permutations per batch element (same pattern as test_setmenunet.py).
   - `options_shuffled = torch.stack([options[0, perm0], options[1, perm1]])`.
   - Forward both with no_grad, un-permute outputs, assert `torch.allclose(out_orig, out_restored, atol=1e-5)`.

5. `test_masking()` -- CSMNET-04:
   - Create CNNSetMenuNet, model.eval().
   - `grid = torch.randn(1, 2, 11, 11)`, `capacity = torch.randn(1, 4)`, `options = torch.randn(1, 10, 6)`.
   - `mask = torch.zeros(1, 10, dtype=torch.bool); mask[0, :5] = True`.
   - Forward with no_grad, assert `out[0, 5:] == 0` all True (padding zeroed), assert `out[0, :5] != 0` any True (real positions non-zero).

6. `test_all_masked()` -- D-19 edge case:
   - Create CNNSetMenuNet, model.eval().
   - `grid = torch.randn(2, 2, 11, 11)`, `capacity = torch.randn(2, 4)`, `options = torch.randn(2, 5, 6)`, `mask = torch.zeros(2, 5, dtype=torch.bool)`.
   - Forward with no_grad, assert `(out == 0).all()`, assert `out.shape == (2, 5)`.

7. `test_save_load()` -- D-13, D-15, D-16:
   - Create CNNSetMenuNet, model.eval().
   - `grid = torch.randn(2, 2, 11, 11)`, `capacity = torch.randn(2, 4)`, `options = torch.randn(2, 6, 6)`, `mask = torch.ones(2, 6, dtype=torch.bool)`.
   - Forward with no_grad, save output as `out_before`.
   - Save to tempfile, create new CNNSetMenuNet, load from tempfile, eval, forward again.
   - Assert `torch.allclose(out_before, out_after, atol=1e-6)`.
