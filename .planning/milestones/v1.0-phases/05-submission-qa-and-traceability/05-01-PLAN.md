# Plan 05-01: Compile, Reference, and Artifact QA

## Goal

Verify all cross-references, citations, artifact paths, and terminology consistency across the full manuscript.

## Tasks

### Task 1: Extract and verify all \ref{} and \label{} pairs

Read all .tex files in `ooh_code/manuscript/sections/` and:
1. Extract all `\label{...}` definitions
2. Extract all `\ref{...}` and `\citep{...}` / `\citet{...}` references
3. Verify every \ref points to an existing \label
4. Report any broken references

### Task 2: Verify all bibliography citations

1. Extract all citation keys from the .tex files
2. Verify each key exists in `references.bib`
3. Report any missing or orphaned bibliography entries

### Task 3: Verify ArtifactTable and ArtifactFigure paths

1. Extract all `\ArtifactTable{...}` and `\ArtifactFigure{...}` paths from the .tex files
2. Verify each file exists at the specified path relative to the manuscript directory
3. Report any missing artifact files

### Task 4: Consistency terminology check

Read all .tex files and check:
1. "RC" usage — is it defined on first use?
2. "MNL" usage — consistent across sections?
3. "ETA" and "IVT" — defined and used consistently?
4. "outside option" vs "outside-option" — consistent hyphenation?
5. "DRT" — defined on first use?
6. Tier labels ("Mechanism diagnostic", "Behavioral stress test", "Descriptive external check") — consistent?

### Task 5: Fix any issues found

Fix all issues discovered in Tasks 1-4. If no issues found, note this in the summary.

## Execution Notes

- This is verification and fix work — read files carefully before editing
- When done, write a SUMMARY.md to `.planning/phases/05-submission-qa-and-traceability/05-01-SUMMARY.md`
