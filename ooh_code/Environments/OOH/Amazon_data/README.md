# Bundled Benchmark Instances

This directory stores the bundled `Austin` and `Seattle` benchmark instances used by the internal work2 simulator.

Each instance folder contains the coordinates and distance data required for the current many-to-one DRT menu experiments.

- For `Austin`, the last 278 locations act as the bundled meeting-point set.
- For `Seattle`, the last 299 locations act as the bundled meeting-point set.
